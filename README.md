## 📋 What is arch-view?

arch-view is a tool designed to show the layout of a Clojure project clearly. It creates an interactive map that helps you understand how files and folders fit together. This tool works on your Windows computer and helps you see the project’s structure without digging through many files.

You do not need to know how to program in Clojure to use arch-view. It guides you by showing the project visually.

## ⚙️ Troubleshooting

If arch-view does not open or crashes:  

- Restart your computer and try again.  
- Make sure you downloaded the Windows version.  
- Check if your Windows is updated to the latest version.  
- Try running arch-view as Administrator: Right-click the icon and select “Run as Administrator.”  

If you have trouble opening your project folder:  

- Confirm it contains Clojure files (look for `.clj` or `.cljs`).  
- Ensure you have permission to view the folder.  
- Avoid network or external drives for best results.
